# Utah Sec Wiki Site

The University of Utah Cybersecurity Club (UtahSec) markdown Wiki.

## Submitting Changes

Please make a pull request with you changes and a UtahSec Officer will review the changes and merge them. 
Thank you to those who have created these docs!

## Local Dev Prerequisites

Follow the instructions in the [Jekyll Docs](https://jekyllrb.com/docs/installation/) to complete the installation of the basic environment. [Git](https://git-scm.com/) also needs to be installed.


## Theme Usage

Please see the [theme's docs](https://github.com/cotes2020/jekyll-theme-chirpy#documentation).
